# SLAE

Программа на C++ для решения систем линейных алгебраических уравнений (СЛАУ).

## Описание

Данный проект представляет собой лабораторную работу, посвящённую решению СЛАУ.  
В программе реализован алгоритм, позволяющий выполнять вычисление решения системы линейных уравнений по введённым исходным данным.

Проект предназначен для практического изучения численных методов, работы с матрицами и организации вычислений на языке C++.  
Результатом работы программы является нахождение решения заданной системы и вывод полученных значений.

## Возможности

- ввод данных для системы линейных уравнений;
- вычисление решения СЛАУ;
- вывод результатов в консоль;
- сборка проекта с помощью `Makefile`.

## Сборка

Для сборки проекта в корневой папке выполните команду:

```bash
make
```

После сборки запустите программу командой:

```bash
main.exe
